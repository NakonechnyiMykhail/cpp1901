1. Создать структуру данных которая отображает контактную форму клиента
какого-либо банка. Структура должна содержать такие данные:
    - ФИО в трех переменных для Фамилии/Имени/Отчества
    - Возраст
    - Дату рождения (создать тип данных date)
    - Почта (создать тип данных email)
    - Имущество (вектор-набор строчных данных) - могут добавляться:машина, квартира и тд.
    - Состояния брака
    - Дети (реализовать через ПЕРЕЧИСЛЕНИЯ создав для примера 5 констант-детей:
        ZERO, ONE, TWO...)
    - Место работы (создать тип данных адрес:
        номер дома, улица, номер квартиры, почтовый индекс, город, страна )
    - Доход (создать тип данных 'деньги' - количество, валюта)
    - Сумма кредита (создать тип данных кредит: значение, годовой процент)

2. Реализовать функцию для расчета ежемесячного платежа в нее передается структура и
    выводится значение
3. Создать вектор-набор клиентов.
4. Создать функцию добавления клиента
5. Создать функцию вывода данных о клиента
6. Создать меню для взаимодействия и выбора действий:
    - вывод
    - добавление и тд.
