без использования пространства имен std

1. Вставить в заголовочный файл функцию рандомных чисел из прошлого дз:
изменить функцию, сделав в ней 2 аргумента:
    1 - начальное значение предела,
    2 - конечное значение предела
Функция возвращает число в заданном пределе


2. реализовать функцию в заголовочном файле,
    1) создать переменную "счетчик" вызовом рандомной функции,
        в пределе от 15 до 20
    2) по этому счетчику будет пробегать цикл, который
        подсчитывает сумму каждого рандомного отрезка длины в
        пределах от 30 до 40 (не включительно).
    3) эта сумма должна возвращаться


3. реализовать в заголовочном файле функцию:
    1) вычисляющую среднюю скорость и возвращающую результат
    2) входящий параметр = время (целое) из основной функции
    3) функция возвращает всю длину деленную на время


4. в основном файле:
    1) подключить заголовочный файл с функцией скорости
    2) объявить константную переменную времени
    3) показать скорость в результате
    4) тело функции должно быть в 2 строки

5. конечный вывод должен быть примерно следующим:
Отчет работы программы=>
___________________________
Количество отрезков = ...
___________________________
Отрезок[1]          = ...
...
Отрезок[n]          = ...
___________________________
___________________________
Общая длина         = ...
Скорость            = ...
